# ChatGPTGamer - Minecraft Multi-User Smart Bot

Features:
- Multi-user bot for Discord + Minecraft Java
- Smart AI Loop with State Machine
- Mining, building shelter, fighting mobs automatically
- Memory for each player: chat history, relationship
- Chat interaction with players in-game
- Logging all actions and chat

Setup:
1. npm install
2. Copy `.env.example` to `.env` and add your DISCORD_TOKEN and OPENAI_API_KEY
3. Run: npm start
